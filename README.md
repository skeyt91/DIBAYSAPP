# DIBAYSAPP

Aplicacion Android para DIBAYS FARDOS.

## Supabase

El esquema inicial esta en `supabase/schema.sql`.

Pasos:

1. Crear un proyecto nuevo en Supabase llamado `DIBAYSAPP`.
2. Abrir SQL Editor.
3. Pegar y ejecutar el contenido de `supabase/schema.sql`.
4. Copiar `Project URL` y `anon public key` para conectar la app.

No subas llaves secretas al repositorio.
